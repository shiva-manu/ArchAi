## Critical Gaps (2)

1. **SSE endpoint burns API calls on client disconnect** — no `req.on('close')` handler. Server keeps iterating through async generator making LLM calls for no one.
2. **Prompt context grows quadratically in build prompt stream** — `promptContext += ...` for each node means 12th prompt includes 11 previous prompts as context.
